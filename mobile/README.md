# Mobile App

Flutter/Dart mobile app frontend

## Tech Stack

- Flutter
- Dart

## Flutter Installation

If using VS Code as the editor (preferred for a Windows dev environment), use the following guide for quick and easy Flutter installation:

https://docs.flutter.dev/install/with-vs-code


## Getting Started

1. Install Flutter
2. Navigate to the root directory of the mobile app: `cd mobile`
3. Run the app: `flutter run -d chrome`
4. Make changes using stateful hot reload. After making changes while app is running, press `r` in the terminal to hot reload and update the running app
